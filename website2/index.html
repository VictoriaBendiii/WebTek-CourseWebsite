    Session Handling

    A session is a technique used to store data from users to be used for future activities of the user within the Web Application.

    With the $_SESSION variable, Session handling is possible with PHP.

    ×
    • A session is a technique used to store data from users to be used for future activities of the user within the Web Application.
    • With the $_SESSION variable, Session handling is possible with PHP.
    • Important functions in Session handling:
      • session_start()
      • isset() - Validation if the current session is running actively or not.
      • unset() - to destroy the session.
    • Web Application generates Session IDs
    • Session IDs are generated after a user has logged in to the Web Application.
    • Session IDs are used for:
      • The web Application to keep track of the user's doings and activities.
      • The web Application to identify the user.
      • The web Application to keep the session of the user secure.
    • Sessions IDs are very prone to attackers that is why itis very essential for the Web Application to secure this.
